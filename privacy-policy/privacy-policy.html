Política de Privacidade

Última atualização: 28/07/2025

Resumo: Esta política explica como o bot Event Hub coleta, usa e protege suas informações quando você interage com ele no Discord.

1. Informações que Coletamos

Dados Automaticamente Coletados:

  • ID do Usuário: Seu ID único do Discord
  • ID do Servidor: ID dos servidores onde o bot está presente
  • Mensagens de Comando: Comandos que você envia para o bot
  • Timestamps: Horário das interações com o bot

Dados Opcionais (apenas se você fornecer):

  • Configurações pessoais (prefixos, configurações de notificação)
  • Conteúdo de mensagens para funcionalidades específicas
  • Tempo de participação em eventos e comandos executados para uso interno da guilda

2. Como Usamos Suas Informações

Usamos os dados coletados para:

  • Funcionalidade do Bot: Processar comandos e fornecer respostas
  • Personalização: Lembrar suas preferências e configurações
  • Melhoria do Serviço: Identificar bugs e melhorar funcionalidades
  • Moderação: Detectar uso inadequado e manter logs de segurança
  • Estatísticas: Gerar estatísticas anônimas de uso

3. Armazenamento e Retenção de Dados

Onde armazenamos:

Seus dados são armazenados em:

  • Os dados são armazenados localmente em arquivos no servidor onde o bot está hospedado
  • Servidores seguros com criptografia

Por quanto tempo:

  • Logs de comando: 90 dias
  • Configurações do usuário: Até remoção manual ou inatividade prolongada
  • Dados de moderação: 120 dias

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Discord: Conforme necessário para funcionalidade da plataforma
  • Requisitos Legais: Se exigido por lei ou ordem judicial
  • Proteção de Direitos: Para proteger nossos direitos legais ou de outros usuários

5. Seus Direitos

Você tem o direito de:

  • Acesso: Solicitar uma cópia dos seus dados
  • Correção: Corrigir informações incorretas
  • Exclusão: Solicitar a remoção dos seus dados
  • Portabilidade: Receber seus dados em formato legível
  • Restrição: Limitar o processamento dos seus dados
Como exercer seus direitos: Entre em contato conosco através dos canais listados na seção de contato abaixo.

6. Segurança dos Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados:

  • Criptografia de dados em trânsito e em repouso
  • Acesso restrito aos dados apenas para pessoal autorizado
  • Monitoramento regular de segurança
  • Backup seguro dos dados

7. Dados de Menores

Não coletamos intencionalmente dados de usuários menores de 13 anos. Se tomarmos conhecimento de que coletamos dados de um menor sem consentimento dos pais, tomaremos medidas para remover essas informações.

8. Cookies e Tecnologias Similares

O bot não usa cookies diretamente, mas pode interagir com a API do Discord que utiliza suas próprias tecnologias de rastreamento conforme sua política de privacidade.

9. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Notificaremos sobre mudanças significativas através de:

  • Atualização da data "Última atualização" no topo desta página
  • Anúncio em nosso servidor Discord de suporte

10. Base Legal (LGPD/GDPR)

Processamos seus dados com base em:

  • Interesse Legítimo: Para fornecer e melhorar nossos serviços
  • Consentimento: Para recursos opcionais que requerem dados adicionais
  • Execução de Contrato: Para cumprir nossos termos de serviço
